# 🌍 Andid HR Yemen

A professional Yemeni platform for job posting and tenders — designed to streamline recruitment, application management, and tender submissions for companies and individuals.

---

## 🧩 Core Features

### 🔹 For Visitors:
- 📝 Browse available jobs.
- 📄 View tender details.
- 📤 Submit job applications with CV upload.
- 📤 Submit tenders with required documents.

### 🔸 For Registered Companies:
- 🎯 Post jobs and tenders.
- 📊 View and manage incoming applications (ATS system).
- 🔐 Secure login via Firebase Authentication.

---

## 🔧 Tech Stack

- **React JS** – Frontend framework.
- **Firebase** – Firestore (database), Storage, and Authentication.
- **Tailwind CSS** – Responsive and clean UI design.
- **React Router** – For page navigation.
- **Netlify / Vercel** – Deployment and hosting.
